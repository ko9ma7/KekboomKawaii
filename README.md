# KekboomKawaii

타워오브판타지 채팅창 미러링 프로그램.

npcap을 필수로 설치할 것.

https://npcap.com/dist/npcap-1.72.exe

윈도우 환경의 경우  winpcap을 설치할 것.

https://www.winpcap.org/install/bin/WinPcap_4_1_3.exe
